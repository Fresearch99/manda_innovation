Most generated outputs are excluded from the public repository.

The analysis scripts write regression tables, summary statistics, advanced-method outputs, placebo diagnostics, and plots to the local output directory configured in `AnalysisConfig`.  These files are derived from restricted source data and can be regenerated by running the analysis pipeline in the appropriate local data environment.

The public repository includes only a small curated set of final figures required to render the overview notebook.  Intermediate regression outputs, full diagnostic folders, placebo results, unit-level estimates, and constructed data files are intentionally not tracked.

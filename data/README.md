This folder is intentionally kept empty in the public repository.

The project relies on patent, inventor, firm financial, and M&A source data that are not redistributed here because they include licensed, restricted, or large external datasets.  The construction scripts in `src/construction/` document how the analysis panels are assembled from the local data environment.

Generated panel files such as `firm_year_panel.pkl`, `final_firm_year_ma_event_study_panel.pkl`, `inventor_ma_event_study_panel.pkl`, and `inventor_year_panel_ma_inventors.pkl` are excluded from the public repository.  They can be regenerated locally by running the construction pipeline with the required source datasets available.
